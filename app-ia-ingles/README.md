# FalaJa English

MVP estatico em `HTML + TailwindCSS + JavaScript` para treino de ingles com foco em fala, escuta e escrita.

## Recursos

- Licoes basicas e intermediarias
- Leitura em voz com Web Speech API
- Reconhecimento de voz no navegador
- Correcao local de escrita e fala
- Progresso salvo em `localStorage`

## Rodar localmente

Abra `index.html` no navegador.

Para melhor suporte de voz, use Chrome.

## Deploy no Vercel

1. Suba estes arquivos para um repositorio GitHub
2. Importe o repositorio no Vercel
3. Framework preset: `Other`
4. Build command: deixe vazio
5. Output directory: deixe vazio
6. Deploy

Como o projeto e estatico, o Vercel publica direto.
